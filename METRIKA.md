# Яндекс.Метрика — polgrek.site

## 1. Создать / взять счётчик

1. Откройте [список счётчиков](https://metrika.yandex.ru/list?period=week&group=day).
2. **Добавить счётчик** (если ещё нет) для домена **polgrek.site** (и при необходимости `www.polgrek.site`).
3. Скопируйте **номер счётчика** (только цифры).

## 2. Вписать ID на сайт

Файл `js/base-config.js`:

```js
window.POL_GREK_METRIKA_ID = 110711984; // активный счётчик polgrek.site
```

Чтобы временно отключить локально: поставьте `0`.

После правки: `git commit` + `git push` (GitHub Pages).

## 3. Рекомендуемые настройки счётчика (кабинет Метрики)

| Опция | Значение |
|--------|----------|
| Принимать данные только с указанных адресов | `polgrek.site`, `www.polgrek.site` |
| Карта кликов | вкл. |
| Вебвизор | вкл. (по желанию) |
| Точный показатель отказов | вкл. |
| Отслеживание хеша | вкл. (якоря `#excerpt`) |
| Электронная коммерция | dataLayer (заготовка) |

Код на сайте уже включает: `clickmap`, `trackLinks`, `accurateTrackBounce`, `webvisor`, `trackHash`.

## 4. Цели (JavaScript-события)

**Метрика → счётчик → Цели → Добавить цель → JavaScript-событие.**

| Идентификатор цели | Когда срабатывает |
|--------------------|-------------------|
| `litres` | Клик по кнопке/ссылке Литрес (`data-track="litres"`) |
| `amazon` | Клик Amazon |
| `telegram` | Клик Telegram (`t.me/polgrekauthor`) |
| `threads` | Клик Threads (`@pol.grek`) |
| `catalog_search` | Поиск в каталоге книг (query ≥ 2 символов) |
| `excerpt_download` | Скачивание отрывка |
| `lang_en` | Переключение на English |
| `lang_ru` | Переключение на Русский |
| `book_view` | Открытие страницы книги |
| `lab_view` | Открытие статьи лаборатории |
| `filter_books` | Клик фильтра в каталоге |

Условия: тип **JavaScript-событие**, имя = идентификатор **точно** как в таблице.

Параметры (в отчётах «Параметры визитов»): `book`, `lang`, `path`, `filter`, `article`.

## 5. Проверка

1. Откройте сайт с ID ≠ 0.
2. Метрика → **Онлайн** — визит.
3. Клик «Литрес» → цель `litres` (через несколько минут в отчёте по целям).
4. Расширение «Яндекс.Метрика» / консоль: нет ошибок `ym is not defined` при ID=0 (скрипт не грузится).

## 6. RU / EN

Один счётчик на весь сайт. Язык уходит параметром `lang=ru|en` и в visit params.
